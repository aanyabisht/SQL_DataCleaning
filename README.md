# 📊 SQL Data Cleaning Project – Layoffs Dataset

## 📌 Overview
This project focuses on cleaning a real-world layoffs dataset containing information about companies across different countries and industries. The raw data contained inconsistencies, missing values, and formatting issues, which were resolved using SQL.

---

## 🎯 Objective
To transform raw and unstructured data into a clean, consistent, and analysis-ready dataset using SQL data cleaning techniques.

---

## 🛠 Data Cleaning Process

The following steps were performed:

- 🔹 Standardized text data (company names, industries, locations)
- 🔹 Converted date columns into a consistent format
- 🔹 Handled NULL and missing values
- 🔹 Removed duplicate records
- 🔹 Dropped unnecessary columns
- 🔹 Cleaned and structured the dataset for further analysis
